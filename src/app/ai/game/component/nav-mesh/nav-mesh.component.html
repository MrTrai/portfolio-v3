Navigation Mesh Generator

This section will introduce on how to generate a navigation mesh for the AI agent to perform path finding. Source Code

What is a Nav Mesh?

Most classic 2D games used the Grid system where the map is divided into a 2D grid/matrix. Nav Mesh is a more optimized version where it produces less meshes compare to the Grid system. Therefore, performing pathfinding is faster on Nav Mesh than on Grid system.

Algorithm

Approach 1: Brute Force DFS

  1. Find all triangles form from all the points in the network
  2. Perform DFS to find the best combination of triangles.
    1. The goal condition is the mesh covers the whole map.
    2. We would also like to pick the mesh where we can merge more triangles and produce more polys > 3 edges.

Approach 2: Greedy

  1. Find all triangles from all the points in the network.
  2. Sort the triangles by area.
  3. Pick the largest triangle that can fit into the mesh until the whole mesh is formed.
    • The goal condition is the mesh covers the whole map.
    • We would also like to pick the mesh where we can merge more triangles and produce more polys > 3 edges.
