What is a Nav Mesh?

Most classic 2D games used the Grid system where the map is divided into a 2D grid/matrix. Nav Mesh is a more optimized version where it produces less meshes compare to the Grid system. Therefore, performing pathfinding is faster on Nav Mesh than on Grid system.

  1. In order for the agent to move from point A to point B, it first finds the 2 path edge corners nearest to A and B, says C and D respectively, it then moves to point C.
  2. The agent now performs a standard A* path finding to D given the network edges. It then follows the path to point D and then move to B.

The main goal of path network is to reduce the search space for path finding. In games where performance is crucial, optimizing path finding run time is more important than ever.
